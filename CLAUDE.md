# CLAUDE.md

This file provides guidance to Kilo Code when reading and writing code.

## Project Overview
This is an interactive GitHub Desktop workflow guide documentation site built with React.

## Tech Stack
- React 18 with TypeScript
- Vite 6 as build tool
- Tailwind CSS v4
- Framer Motion for animations
- Lucide React icons

## Code Review Guidelines
- Review React components for proper TS typing
- Check accessibility in documentation components
- Verify Tailwind class usage follows design system
- Ensure animations are performant

## Commands
- `npm install` - Install dependencies
- `npm run dev` - Start development server
- `npm run build` - Build for production
- `npm run lint` - Run ESLint
